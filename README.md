KEYGEN
==========

Keygeneration algorythm for arduino and Raspberry pi


Usage
=====

### Raspberry  pi
install
```
sudo make install
cd examples_RPi
make
```

What to do after changes to the library
```
sudo make clean
sudo make install
cd examples_RPi
make clean
make
```

What to do after changes to a sketch
```
cd examples_RPi
make <sketch>

or 
make clean
make
```

How to start a sketch
```
cd examples_RPi
sudo ./<sketch>
```
